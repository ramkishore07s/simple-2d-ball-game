### CONTROLS:
|Key         |Action                                  |
|:-------------------------------|:---------------------------------------|
|  A         |      Move left                         |
|  D         |      Move right                        |
|  Space     |      Jump                              |  
|  Left      |      Pan left                          |
|  Right     |      Pan right                         |
|  Up        |      Pan up                            |
|  Down      |      Pan down                          |
|  Scroll    |      Zoom in/out                       |
|  G/H       |      Toggle automatic pan & zoom       |



### COMPILING:
cd into the cloned folder and
```
mkdir build
cd build/
cmake ..
make 
```



### EXECUTING:
The executable will be located at the “simple-2d-ball-game/build/“ folder with the name “graphics_asgn1”.
